# Updating H1-Mod

You should get a message when you open the game that says the same as the image below. Press **Yes** to get the latest update.

![](/img/updater.png)

* If this message doesn't show, all you need to do is to [uninstall H1-Mod](h1-uninstall) and [reinstall H1-Mod](h1-install#installing-the-h1-mod-client).
