# IW7-Mod Changelog

This is a list of updates IW7-Mod has, and the changes through the updates. New updates will be automatically installed when you launch IW7-Mod.

## v1.0.2

### Changes
- Fixed Spaceland door prices being inaccurate for solo players on dedi
- Fixed dvar flags not being overrided correctly (ex: deadzone dvars work now)
- Fixed timescale not being replicated
- Fixed Discord RPC not showing the map being played
- Reduce the stress of profile info packets [iw7-mod#634](https://github.com/auroramod/iw7-mod/pull/634)

### Known Issues

- Missions will level up, but rewards are not given for them [iw7-mod#618](https://github.com/auroramod/iw7-mod/issues/618)
- Dedicated servers may experience little stutters on players joining and disconnecting above 12 real players in a lobby
- Some weapons aren't being given ingame properly

## v1.0.1

### Changes
- Fixed bad dvar resulting in glitched match ends on dedis [iw7-mod#620](https://github.com/auroramod/IW7-MOD/issues/620)
- Fixed quartermaster not knowing your free supply drop amount [iw7-mod#617](https://github.com/auroramod/IW7-MOD/issues/617)
- Fixed "status" command crashing during map rotation
- Fixed Zombies supply drops taking keys and giving no crate
- Fixed "Genesis Dusk" map not being shipped on Release builds
- com_maxfps is now 1000 max again
- cg_chatHeight & cg_ChatTime are now saved dvars
- Increased server timeout count
