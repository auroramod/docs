Creating a IW7 Dedicated Server

To create a "dedicated" server, you must have a Computer or VPS that is constantly online 24/7 with good internet connection. It is recommended to use a VPS that is not your machine and home connection hosting it. If you are simply hosting a local server you turn on and off, this guide will work perfect for you.

You may need to Port Forward your server for people to see it in the Server Browser.

Requirementsâ€‹

  • Redistributables required for Call of Duty
  • A text editor (VSCode or Notepad++ is recommended)
  • A valid installation of Call of DutyÂ®: Infinite Warfare from Steam

Setting up your serverâ€‹

warning

The server files have not been optimized and perfected yet. The game does require around ~90GB of space, so be sure to have some storage allocated to setup your server.

  1. Download the IW7 Dedicated Server Resources from GitHub. You can do so by clicking the green Code dropdown box, and clicking Download ZIP on the bottom.
  2. Move the iw7-mod folder and the .bat files into your Infinite Warfare folder.
  3. Port Forward the UDP port(s) your launcher scripts will use. By default, it is 27016. To learn more about Port Forwarding, click here.
  4. Now you can edit your configs and edit your server launcher scripts!

Editing your server configurationâ€‹

You can edit the .cfg files in the iw7-mod folder to your liking. This includes settings like:

  • server name (hostname)
  • gametype
  • maps
  • MOTD (Message of the Day)

Editing your server launcher scriptsâ€‹

You can edit the .bat files in your Infinite Warfare directory to your liking also. This includes settings like:

  • mod (fs_game)
  • port
  • developer script (for debugging)
